================================================================================
                        🚀 部署與運行
================================================================================

【前端】
- 框架: Flutter 3.0+
- 構建命令: flutter build web --release
- 輸出位置: build/web/
- 部署目標: GitHub Pages / 靜態伺服器

【後端】
- 框架: Node.js + Express
- 啟動命令: npm run dev（開發）/ npm start（生產）
- 監聽埠: 3000
- 環境變數: HUGGING_FACE_API_TOKEN（必需）

【快速啟動】
1. 後端: cd backend && cp .env.example .env && npm install && npm run dev
2. 前端: flutter pub get && flutter run -d chrome
3. 應用訪問: http://localhost:8080

================================================================================
                      🎯 主要功能說明
================================================================================

1【時間視覺化】
   - 每個任務區塊的高度反映實際時長
   - 24 小時完整視圖配參考線
   - 一眼看出時間規劃的合理性

2【拖曳功能】
   - 任務池 ↔ 時間表無縫拖曳
   - 拖曳下邊緣精確調整時長（0.5hr 單位）
   - 視覺反饋完整（把手、透明度、色彩變化）

3【任務編輯】
   - 點擊任務打開編輯對話框
   - 修改名稱、時長、備註
   - 實時保存到本地存儲

4【完成追蹤】
   - Checklist 式完成標記
   - 完成時顯示慶祝動畫 🎉
   - 完成任務視覺上有區別（灰色 + 刪除線）

5【倒數計時】
   - 重大事件倒數卡片
   - 自動計算剩餘時間
   - 快速管理（新增 + 刪除）

6【AI 助手】
   - ✨ 按鈕打開 AI 分解對話框
   - 輸入複雜任務描述
   - AI 幫你拆解成子任務 + 時間估算

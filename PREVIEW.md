# Tech Vibe 網頁預覽指南 - v2

## 🚀 新版本特性

### 主要佈局變更
- **左側**：月曆選擇器（日期選擇）
- **右側**：時間表網格（0.5 小時時段）

### 拖曳機制升級
- ❌ 舊版本：區塊拖曳（不符合需求）
- ✅ **新版本**：時間表式排程
  - 每 0.5 小時為一個時段
  - 可直接插入任何時間段的任務
  - 高度與時長自動匹配

## 🎯 如何使用

### 1️⃣ 選擇日期
- 在左側月曆中點擊日期
- 選中的日期會高亮顯示（藍色背景）
- 右側會顯示該日期的日程表

### 2️⃣ 新增任務
在左側「新增任務」區：
- **任務名稱**：輸入任務描述
- **開始時間**：選擇 00:00 - 23:30（每 30 分鐘一個選項）
- **時長**：選擇 0.5 到 4 小時
- 點擊「新增任務」按鈕

### 3️⃣ 查看日程
- 右側顯示完整的 24 小時時間表
- 任務自動按開始時間排列
- 區塊高度與時長對應（0.5h = 30px）
- 上方顯示該天的總規劃時數

### 4️⃣ 管理任務
- **刪除**：懸停任務，點擊右上角「×」按鈕
- **拖曳**：任務塊可拖曳（未來支援跨日期移動）

## 📊 時間網格說明

```
時間標籤              時程表（24 小時）
├─ 00:00 ┐           ┌─────────────────┐
│        │ 0.5h      │                 │
├─ 00:30 ┤           │  [任務 A: 1h]   │
│        │           │                 │
├─ 01:00 ┘           ├─────────────────┤
│                    │ [任務 B: 0.5h]  │
├─ 01:30             ├─────────────────┤
│                    │                 │
├─ ...               │                 │
│                    └─────────────────┘
└─ 23:30
```

### 網格特性
- 每個槽位高度：30px（代表 0.5 小時）
- 1 小時任務：60px 高
- 2 小時任務：120px 高
- 自動對齐至 30 分鐘邊界

- ✅ 支援多日期管理

## 💡 使用技巧

### 快速添加常用任務
1. 先填入任務名稱
2. 選擇常用時間和時長
3. 點新增
4. 重複步驟 1-3（名稱會保留）

### 查看不同日期
- 使用月曆快速跳轉
- 左上方箭頭切換月份
- 點擊月份的任何日期即更新右側

### 管理密集時段
- 超過 16 小時會有視覺提醒
- 可滾動查看完整 24 小時
- 任務自動按時間排列，易於調整

# Tech Vibe 網頁預覽指南

## 🚀 快速開始

### 1. 訪問預覽版本
伺服器已在後台運行，訪問：
```
http://localhost:8000
```

### 2. 測試功能

#### 📝 新增任務
- 在頂部「新增任務」區輸入任務名稱
- 選擇預估時長（0.5 - 4 小時）
- 按「新增」按鈕或 Enter 鍵
- 新任務會出現在左側「待放置的任務」區

#### 🎯 拖曳管理
- 從左側任務區塊拖曳到右側「今日規劃」
- 拖曳時會看到淡化效果和懸停反饋
- 放開後任務會堆疊在容器中

#### 🗑️ 刪除任務
- 在任務區塊右上角找到「×」按鈕
- 點擊確認刪除

#### 📊 視覺參考
- 右側容器有虛線標示：8h、16h、24h
- 區塊高度與時長成正比（1h = 30px）
- 任務超過 16h 會出現警告提示

## 💾 本地儲存

所有資料自動保存到瀏覽器 localStorage：
- 刷新頁面不會丟失任務
- 關閉瀏覽器資料仍會保留
- 清除瀏覽器快取會刪除資料

## 🔄 與 Flutter 版本的關係

此 HTML/CSS/JS 版本是 Flutter Web 版本的：
- **UI 參考**：按鈕、顏色、佈局相同
- **功能驗證**：所有交互邏輯已實現
- **測試基礎**：可用來驗證 Flutter 實現

如果要轉換為 Flutter Web：
1. 確認有 Flutter SDK ≥ 3.0.0
2. 運行 `flutter pub get`
3. 運行 `flutter run -d chrome`

## 📋 檔案說明

### index.html (2.6 KB)
HTML5 語義化結構，包含：

- 任務建立區
- 待放置任務池
- 今日規劃容器

CSS3 樣式和動畫，包含：

- Hover 和拖曳視覺反饋
- 響應式媒體查詢
- 光滑的過渡動畫

- HTML5 拖曳 API 整合
